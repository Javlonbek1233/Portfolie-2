## 🏗 Vercel'ga Joylash (Vercel Deployment Settings)

Agar ushbu loyihani Vercel platformasiga yuklayotgan bo'lsangiz, Vercel sizdan so'raydigan sozlamalarni quyidagicha belgilang:

| Sozlama nomi (Setting Field) | Qiymati (Value) |
| :--- | :--- |
| **Framework Preset** | **Vite** (yoki *Other* deb tanlab quyidagilarni yozing) |
| **Build Command** | `npm run build` |
| **Output Directory** | `dist` |
| **Install Command** | `npm install` |

### Joylash bosqichlari (How to deploy):
1. Loyihani o'zingizning GitHub profilingizga (*Github: https://github.com/Javlonbek1233*) yuklang.
2. [Vercel](https://vercel.com) saytiga kiring va "Add New" -> "Project" tugmasini bosing.
3. O'zingizning GitHub repo-ingizni import qiling.
4. Yuqorida ko'rsatilgan **Vite** sozlamalari avtomatik tanlanadi, agar tanlanmasa qo'lda kiriting va **"Deploy"** tugmasini bosing! 🎉

## 💻 Mahalliy Kompyuterda Ishga Tushirish (Local Setup)

Loyiha kodlarini kompyuteringizda ishga tushirish uchun quyidagi buyruqlarni ketma-ket bajaring:

1.  **Kutubxonalarni o'rnatish:**
    ```bash
    npm install
    ```

2.  **Dasturni ishga tushirish (Development Mode):**
    ```bash
    npm run dev
    ```
    *Dastur brauzerda `http://localhost:3000` yoki terminalda ko'rsatilgan manzilda ochiladi.*

3.  **Loyihani Vercel yoki ishlab chiqarish uchun tayyorlash (Production Build):**
    ```bash
    npm run build
    ```
